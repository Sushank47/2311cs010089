# Campus Notification Platform

A MERN stack application designed to address the campus notification problem where students receive real-time updates regarding **Placements**, **Events**, and **Results**.

## Project Roadmap

This repository contains the incremental progress of the platform across multiple stages:

*   **Stage 1:** [notification_system_design.md](file:///stage1/notification_system_design.md) - REST API contract design, JSON schema definitions, and real-time push mechanism design.
*   **Stage 2:** Requirement Analysis & User Stories.
*   **Stage 3:** Project System Architecture & MVC Mapping.
*   **Stage 4:** UAT Plan & Testing Strategies.
*   **Stage 5:** Consolidated Architecture Manifest.
*   **Stage 6:** Full-stack implementation (HTTP Baseline).
*   **Stage 7:** Full-stack implementation (Real-time Socket.io updates).
